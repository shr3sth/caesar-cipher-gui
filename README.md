# Caesar Cipher Explorer

A real-time Caesar Cipher GUI application built with Python and Tkinter.

## Features

- Real-time encryption and decryption
- Interactive shift slider (-25 to +25)
- Multi-line text input
- Multi-line output display
- Copy output to clipboard
- Clear input and output
- Dark-themed interface
- Linux compatible

## Technologies Used

- Python 3
- Tkinter
- Git
- GitHub

## How It Works

The Caesar Cipher is a substitution cipher that shifts each letter in the alphabet by a specified number of positions.

Examples:

Shift = 3

HELLO

↓

KHOOR

Shift = -3

KHOOR

↓

HELLO

## Installation

Clone the repository:

```bash
git clone https://github.com/shr3sth/caesar-cipher-gui.git
```

Move into the project directory:

```bash
cd caesar-cipher-gui
```

Run the application:

```bash
python3 main.py
```
